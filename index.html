Learn The Fundamental

Frond-end web development, also known as client-side development is the practice of producing HTML, CSS and Javascript for a website or Web application so that a user can see and interact with them directly

We will be learning technologies such as HTML, CSS, Bootstrap, SAAS, Javascript, Jquery,and React. All along, we will be working on real life projects to apply all we are learning.

Learn Back-end Development

Back end development refers to the server side of an application and everything that communicates between the database and the browser. Here you are primarily focused on how the site works.

We will be learning NodeJS, Express with NodeJS, Application Programming Interfaces(API), Databases, SQL, MongoDB, Mongoose. All along, we will be working on real life projects to apply all we are learning.
